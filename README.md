## Description

Simple API Rest with NestJS for create library catalogue of books, based on the "NestJS: criando uma API Rest com TypeScript" course by Alura.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
